Agent Skills

把一次性提示词变成可复用流程

Skills 用来包装稳定工作流：说明、参考资料、资源和可选脚本。它适合重复任务，也适合和自动化搭配。

OpenAI Skills 文档看插件关系

推荐 Skill 方向

先做用户最常用的 6 类工作流

01

安装排障 Skill

统一处理 Windows SmartScreen、macOS Gatekeeper、Codex 检测失败和日志定位。

02

报告与文档 Skill

把资料整理、报告结构、引用检查和最终输出做成稳定工作流。

03

GitHub Review Skill

读取 PR、CI、评论和代码上下文，优先定位 bug、风险和缺失测试。

04

浏览器验收 Skill

打开本地或线上页面，截图检查按钮、表单、跳转和响应式布局。

05

表格和数据 Skill

清洗表格、生成公式、做指标快照，并把结论变成可分享说明。

06

团队模板 Skill

把团队常用 SOP、交付规范和检查清单放进 repo 或个人技能目录。

什么时候该写

Skill 不等于提示词收藏夹

会重复

同类任务每周、每天或每个项目都会出现。

有步骤

流程顺序、检查项和输出格式相对稳定。

要资料

需要引用说明书、模板、脚本、样例或行业术语。

要分发

希望自己、团队或客户都能按同样方式执行。
