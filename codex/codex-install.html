安装后下一步

把 Codex 接到 LinaryAI

Codex 官方 App 装好之后，还需要把 LinaryAI 的 API 地址和令牌写进去。你可以任选下面一种方式。

方式一：LinaryAI 安装助手

回到安装助手页面，下载对应系统的 LinaryAI 配置工具，输入令牌后自动写入配置和插件包。

方式二：CC Switch

在 LinaryAI 令牌管理里点击 CC Switch，一键导入 Codex 供应商配置。

API 地址

确认 Codex 或 CC Switch 中使用的是 https://api.linaryai.top/v1。

打开 LinaryAI 安装助手 查看 CC Switch 教程 获取 LinaryAI 令牌
