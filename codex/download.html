Windows 图文安装教程

从下载安装到打开 Codex

Windows 版直接提供 EXE 安装包。用户只需要下载、双击、输入 LinaryAI API Key。

下载 Windows EXE

以下图片为按当前 LinaryAI 安装器流程制作的界面示意图；正式发布前建议替换为测试机真实截图。

1. 下载 Windows 安装包

点击“下载 Windows 版”，浏览器会下载一个 EXE 文件。

2. 双击打开 EXE

如出现 SmartScreen 提示，点击“更多信息”后选择“仍要运行”。

3. 检测 Codex 并准备 Key

检测不到时，点击“下载 Codex”，安装完成后再点“重新检测”。

4. 输入 LinaryAI API Key

粘贴完整 Key，然后点击“下一步”。不需要手动打开 Terminal。

5. 等待自动配置

安装器会写入 LinaryAI API 配置、安装插件包，并设置 Codex 简体中文。

6. 完成安装

点击“完成”退出向导，之后 Codex 会使用 LinaryAI 配置和中文体验。

如果 Windows 提示“不明发布者”，这是未购买商业代码签名证书时的常见系统提示。

macOS 图文安装教程

解压后用 Setup App 配置

下载 ZIP 后解压，双击 LinaryAI Codex Setup.app。若电脑还没有 Codex，安装器会提示打开官方 Codex 下载页。

下载 macOS ZIP

以下图片为按当前 macOS Setup App 脚本流程制作的界面示意图；正式发布前建议替换为测试机真实截图。

1. 下载 macOS ZIP

点击“下载 macOS 版”，下载 LinaryAI Codex 的 ZIP 压缩包。

2. 解压并打开 Setup App

正常情况双击 LinaryAI Codex Setup.app。若系统拦截，再运行放行脚本。

3. 如果没有 Codex，先安装 Codex

没有检测到 Codex 时，点击“打开 Codex 下载页”。

4. 输入 LinaryAI API Key

在系统弹窗里粘贴 API Key，确认无误后点击“开始配置”。

5. 完成并打开 Codex

看到配置完成提示后，Codex 会使用 LinaryAI API、插件包和简体中文设置。

如果 macOS 提示“来自不明开发者”，先运行包内放行脚本，或在系统设置的隐私与安全性里选择“仍要打开”。
