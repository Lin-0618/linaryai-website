# LinaryAI Website

Static export of the current `https://linaryai.top/` website.

## Contents

- Main LinaryAI / Lin.Build landing page
- Codex installer page under `/codex/`
- Public static assets and current downloadable installer files

## Deployment

This repository is intended to be deployed through GitHub Pages from the repository root.

The production `https://linaryai.top/` server is still separate unless DNS or web server routing is changed manually.
